# project3

## Project Description

The application will allow the end-user to dynamically create a custom form. The user can select from a variety of input by dragging and dropping elements to create a desired form only limited to their imagination. Some of the form elements include text, file, date, selection, color, text area, button, icon, paragraph, radio, checkboxes, and numeric slider. After the form is built this form can be exported in a JSON.
